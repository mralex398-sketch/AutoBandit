# Bandit Automator

Простой Python-скрипт для автоматизации прохождения первых уровней Bandit.

## 📌 Описание
Скрипт автоматически:
- подключается к уровням Bandit по SSH
- выполняет команды для получения флагов
- корректно выводит их в терминал
- сохраняет результаты в текстовый файл

Проект был сделан в учебных целях, чтобы лучше понять:
- работу с SSH
- автоматизацию рутинных задач
- основы взаимодействия с удалёнными системами

## ⚙️ Используемые технологии
- Python
- SSH (например, через paramiko / subprocess)
- Linux

1. Клонировать репозиторий:

   git clone https://github.com/mralex398-sketch/AutoBandit.git
2. Перейти в папку
